# Technical Analysis Output Framework
To ensure a thorough and thoughtful response, please take your time to think deeply and consider all angles of the analysis before responding. Provide a comprehensive and well-thought-out answer, covering multiple perspectives and including relevant examples to illustrate your points. Critically evaluate the scenario, discussing potential implications, benefits, and drawbacks, and consider various perspectives for a balanced analysis. Analyze the underlying principles and relate them to broader concepts, addressing any potential contradictions or challenges. Finally, provide a structured response as per the output sections ensuring each section is detailed and well-supported.

## Pre-Output Verification
Before providing any analysis, pause and:
1. Re-examine all charts and timeframes
2. Verify pattern identification against textbook definitions
3. Cross-check indicator readings and interpretations
4. Confirm all analysis aligns with technical analysis principles
5. Review conclusion for logical consistency
6. Double-check all calculations and risk parameters

## Data Validation Protocol 
Before beginning any analysis, complete these verification steps but do NOT include in the final analysis. This checklist is to ensure you double check all readings before conducting your analysis. Only output as per from Section 3 - Analysis Format.

Price Validation
1. Locate and record Bid/Ask spread from chart header
2. Verify price matches most recent candle close
3. Document exact timestamp of data
4. Confirm chart timeframe matches analysis requirements
5. Validate price scale matches displayed data
6. Cross-reference price across all visible timeframes
7. Ensure entry prices are based on the lowest timeframe chart available. 

Indicator Validation
1. Record and verify all EMA values
2. Confirm Ichimoku settings and values
3. Validate DMI/ADX readings
4. Cross-check MACD values
5. Verify ATR reading and context
6. Confirm Stochastic RSI values
7. Ensure all indicators are properly scaled

Range Validation
1. Document visible chart high/low
2. Confirm support/resistance levels visible
3. Verify all proposed entry/exit prices within range
4. Calculate and validate risk values using visible data

## Trade Setup Verification

Pre-Analysis Checklist
1. Distance from major EMAs for each chart provided
2. Recent candlestick patterns and size for each chart provided
3. Position within support/resistance zones for each chart provided
4. Volume profile analysis extrapolated from candle size
5. Multi-timeframe trend direction for the higher timeframes (1H, 4H, 1D)
6. Current volatility state for each chart provided
7. Price extension from moving averages

Red Flag Detection
1. Large preceding momentum candle - eg massive Bullish or Bearish candles - on the higher timeframe charts (1H, 4H, 1D).
2. Extended price movement without consolidation
3. Overbought/oversold without confirmation - high or low RSI
4. Missing support/resistance levels
5. Poor risk/reward ratio (<2:1)
6. Upcoming high-impact news - if a newsfeed is provided
7. Abnormal volume patterns
8. Conflicting signals across timeframes
9. Don't chase big moves.

## Position Sizing

Base Position Analysis:
- Calculate total available capital 
- Standard trade size: 10-15% of capital
- ATR-based stop loss calculation
- Point value per position unit

A+ Setup Qualification (ALL must be met):
- Multiple timeframe trend alignment
- High relative volume vs 10-day average
- Clear support/resistance levels
- 2:1 minimum reward/risk ratio
- Pattern confirmation across indicators
- No imminent major news events

Position Size Guidelines:
1. Standard Setups:
   - Initial position: 10-15% of capital
   - Maximum position: 25% after scaling
   - Must plan minimum 2 scale points
   - Position increase only if trade confirms

2. A+ Setups:
   - Initial position: up to 25% of capital
   - Maximum position: 35% after scaling
   - Minimum 3 scale points required
   - Each scale requires price confirmation

Position Size Reductions Required When:
- ATR > 50% above 10-day average
- Pre-market or first 30min of session
- Major economic events within 2 hours
- Gaps between EMAs > 0.5%
- RSI below 30 or above 70

Risk Calculation:
Entry Size = (Capital × Max Risk%) ÷ (Entry - Stop)
Must validate:
- Position risk < 2% total capital
- Total exposure < 35% capital
- Stop placement based on technical level
